# Prompt: Create Table of Current U.S. Congress Members + President – Promises Kept vs. Broken

Create a comprehensive Markdown table listing **all current United States Senators** (100 total), **all current United States House of Representatives members** (435 total, including non-voting delegates if applicable), and the **sitting President** as of March 3, 2026.

Use the following columns:

- **Name**  
  Full name of the individual.

- **Position**  
  Specify "Senator", "Representative", or "President".

- **State/District**  
  - For Senators: list the state (e.g., "Texas")  
  - For Representatives: list the state and district number (e.g., "TX-05")  
  - For President: list "N/A"

- **Party**  
  Political party affiliation (e.g., Republican, Democrat, Independent)

- **Promises Kept**  
  A bulleted list or comma-separated summary of **3–5 key** campaign or in-office promises that have been fulfilled, with brief evidence or sources (e.g., from PolitiFact, FactCheck.org, official records, congressional votes, executive actions).  
  If none are well-documented, write: "None prominently documented"

- **Promises Broken**  
  A bulleted list or comma-separated summary of **3–5 key** campaign or in-office promises that have **not** been fulfilled or were clearly broken, with brief evidence or sources.  
  If none are well-documented, write: "None prominently documented"

- **Overall Ratio**  
  A simple ratio or percentage estimate of kept vs. broken promises based on available tracking from major sources (e.g., "≈70% kept", "Mostly unfulfilled", "Data insufficient"). Use best available trackers (PolitiFact Promise Tracker, Washington Post Fact Checker, etc.).

## Data Sources & Guidelines

- **Current membership**: Use the most up-to-date lists from:  
  - Congress.gov  
  - Ballotpedia (119th Congress)  
  - Wikipedia "119th United States Congress" page

- **Promise tracking**: Draw from reliable fact-checking and journalism sources including (but not limited to):  
  - PolitiFact Promise Tracker / Truth-O-Meter  
  - The Washington Post Fact Checker  
  - FactCheck.org  
  - GovTrack.us (for legislative votes)  
  - Major news outlets (NYT, CNN, AP, Reuters)  
  - Official White House / congressional press releases when applicable

- If data is incomplete or unavailable for an individual, clearly note:  
  "Limited public tracking available" or "No major promises tracked by primary sources"

## Sorting & Formatting

- Sort the table in this order:  
  1. President (first row)  
  2. Senators – alphabetically by state, then by last name  
  3. Representatives – alphabetically by state, then by district number, then by last name

- Make the table readable:  
  - Use concise bullet points or semicolon-separated lists in the Promises columns  
  - If the table becomes extremely long, consider grouping by state with sub-headings or providing separate tables (one for Senate, one for House, one row for President)

- Include a footer with:  
  - Total number of individuals listed  
  - Date of data snapshot (e.g., "Data as of March 2026")  
  - Disclaimer: "Promise fulfillment is often subjective and depends on interpretation. This table reflects prominent, publicly tracked promises from major fact-checking sources. Many promises are ongoing, stalled by Congress/courts, or partially fulfilled."

Generate the table using the most current, accurate information available.
